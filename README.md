# FPGA-Audio-SRC: 基于 FPGA 的高保真音频采样率变换系统 🎵
📖 项目简介
本项目是一个基于 FPGA 的数字音频采样率变换（SRC, Sample Rate Converter）系统。旨在解决不同音频标准（如 CD 机的 44.1kHz 与专业音频的 48kHz）之间的数字交互问题。

系统核心采用分段线性插值算法实现非整数倍采样率转换，并通过级联 64阶 FIR 低通滤波器有效抑制了插值引入的高频镜像噪声与混叠信号。结合外围的 I2S 编解码模块，实现了端到端的高保真音频实时采集、重采样与还原输出。

✨ 核心特性
核心算法：基于“相位累加器”架构的跨时钟域分段线性插值（44.1kHz 升采样至 48kHz）。

抗混叠滤波：内置基于汉宁窗设计的 64 阶 FIR 低通滤波器（截止频率 20kHz），平滑波形并消除高频频域假影。

硬件接口驱动：纯 Verilog 编写的标准 I2S 总线全双工驱动协议。

跨时钟域处理：采用 MMCM 高精度分频与双异步 FIFO 架构，完美解决多时钟域（44.1k/48k/128k）的数据传输冲突与亚稳态问题。

🛠️ 硬件与开发环境
主控芯片：标准 FPGA 开发板（系统时钟频率需满足要求，本项目使用内部时钟管理单元生成 11.2896MHz 和 30.72MHz）

ADC 音频采集模块：PCM1808 (采样率配置为 44.1kHz，I2S 从模式)

DAC 音频解码模块：PCM5102A (采样率配置为 48kHz，I2S 从模式)

开发软件：Vivado / Modelsim (用于逻辑综合与仿真)

验证工具：MATLAB (用于时频域 FFT 分析与 THD 计算)

📊 性能表现 (测试数据)
利用 MATLAB 对 FPGA 仿真的 1kHz 正弦波数据进行 FFT 频谱分析与总谐波失真 (THD) 计算验证：

未经 FIR 滤波（仅线性插值）：由于截断效应与频谱泄露，总谐波失真 (THD) 约为 -38.34 dB。

经过 FIR 滤波优化后：带外高频混叠得到极好压制，THD 显著降至 -62.31 dB (约 0.076%)，信纳比 (SINAD) 大幅提升，完全满足高保真音频重采样需求。

🚀 快速上手说明
仿真验证：

将 Verilog 文件导入 Modelsim 或 Vivado 中。

运行 top_tb.v，仿真将自动生成处理前后的音频数据 txt 文件。

可将 txt 数据导入 MATLAB 运行分析脚本验证波形。

下板运行：

按照代码中定义的时钟频率，在 Vivado 中例化相应的 MMCM (PLL) 和 FIR Compiler IP 核。

连接 PCM1808 和 PCM5102A 模块。

分配引脚并生成 Bitstream 烧录至 FPGA。使用信号发生器输入 1kHz 正弦波，示波器观察输出波形即可。
